# trepan-ni

A gdb-like debugger in the trepan debugger family. This code
is based on (and largely made up of) the nodejs debugger.

## Setup:
```bash
$ npm install
```

## Run

```
Usage: trepan-ni script.js
       trepan-ni <host>:<port>
```
